    # fdisk分区不超过2T的磁盘

    # 不超过2T的磁盘都可以使用fdisk命令进行分区，如果超过2T，用fdisk每次最多也只能分2T的空间

    # 1、首先用fdisk -l发现待分区的磁盘 /dev/sdb

    # fdisk -l

    # 2、fdisk /dev/sdb对该磁盘进行分区，输入m并回车

    # fdisk /dev/sdb

    # 3、输入n并回车，n是“new”新建分区的意思

    # 4、输入p并回车

    # 5、输入数字1并回车

    # 6、采用默认值一直回车会将整个20G都分给/dev/sdb1

    # 7、输入w  "write"并回车，意思是对刚才的结果进行保存

    # 8、再次使用fdisk -l查看分区的结果

    如图分的新区为/dev/sdb1,，创建的新区格式化后就可以挂载使用了

    # reboot
